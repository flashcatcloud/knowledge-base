## 访问个人中心
---

将鼠标悬停在左下角用户图标上，选择 **基本信息** 或点击 **平台管理**。

![个人中心入口](https://fcpub-1301667576.cos.ap-nanjing.myqcloud.com/flashduty/doc/zh/fd/gerenzhongxin-1.png)

## 基本信息
---

| 配置项 | 说明 |
| :--- | :--- |
| **账号身份** | 主体账号或成员账号 |
| **昵称** | 支持中英文，用于界面显示 |
| **邮箱/手机号** | 修改时需验证码验证 |

支持的手机号地区，请参考[外呼固定号码列表](https://docs.flashcat.cloud/zh/flashduty/frequently-asked-questions)。

## 消息通知
---

按告警级别配置不同的通知方式。

:::tip
仅当满足以下条件时，才会按个人偏好通知：
- 故障直接分派到个人，或
- 分派策略设置为"单聊 + 遵循个人偏好"
:::

如需使用钉钉、微信、飞书等 IM 通知，请先在对应应用中完成账号关联。

## APP Key
---

APP Key 用于 API 请求认证。

| 限制 | 说明 |
| :--- | :--- |
| **数量上限** | 每个账号最多 5 个 |
| **权限范围** | 拥有全部 API 操作权限 |
| **安全提示** | 仅创建时显示，请妥善保存 |

:::caution
- APP Key 泄露可能导致数据安全风险，请务必保密。
- 删除前确认无业务依赖，删除后引用该 Key 的业务将立即失效。
:::

## 常见问题
---

<details>
<summary>设置了个人偏好通知方式，为什么没有触发通知？</summary>

1. 检查协作空间是否配置了分派策略。未配置分派策略时，故障不会触发通知。
2. 检查分派策略的单聊渠道设置。仅"遵循个人偏好"模式下才会使用您的个性化配置；"遵循统一设定"模式下将忽略个人偏好。

进入 协作空间详情 → **分派策略** 查看具体设定。

</details>
