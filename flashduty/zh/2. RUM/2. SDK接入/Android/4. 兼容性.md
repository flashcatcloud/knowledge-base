# Android 实时用户监控（RUM）— 版本兼容性说明

本文档说明 Android RUM SDK 所支持的 Android 系统版本、平台及开发环境。

---

## 支持的 Android 系统版本

| 类别                  | 支持范围                       |
| --------------------- | ------------------------------ |
| **最低 Android 版本** | Android 6.0（API level 23）    |
| **最高 Android 版本** | 当前最新 Android 版本          |
| **支持的设备类型**    | Android 手机、平板、Android TV |

> 不支持 Android 6.0（API level 23）以下的系统版本。

---

## 支持的平台

Android RUM SDK 支持以下平台：

- Android 手机应用
- Android 平板应用
- Android TV 应用

---

## 支持的开发语言

| 开发语言 | 是否支持 |
| -------- | -------- |
| Java     | ✅       |
| Kotlin   | ✅       |

---

## SDK 版本兼容性

| SDK 主版本           | 支持的 Android API |
| -------------------- | ------------------ |
| **v3.x（当前版本）** | API 23+            |
| v2.x                 | API 23+            |
| v1.x                 | 已废弃             |

> 已废弃版本不建议用于新的集成项目。

---

## 构建与工具链要求

| 要求项       | 说明                           |
| ------------ | ------------------------------ |
| **AndroidX** | 必须使用                       |
| **构建系统** | Gradle                         |
| **Kotlin**   | 需与 AndroidX 生态版本保持兼容 |

---

## 功能兼容性说明

### Android TV

Android TV 应用与普通 Android 应用具有相同的最低系统版本要求。

---

### Jetpack Compose

- 支持 Jetpack Compose 的埋点与监控能力。
- 具体兼容性取决于应用所使用的 Compose 版本。

---

### WebView 监控

- 支持 WebView 监控功能（需显式开启）。
- 兼容性取决于系统 WebView 版本。

---

### 第三方库集成

提供对常见 Android 库（如 OkHttp、Timber、Glide 等）的集成支持，其兼容性取决于对应第三方库本身的系统要求。

---

## 已废弃支持

- 不支持 Android 6.0（API level 23）以下系统版本。
- 已废弃的 SDK 版本可能不再提供功能更新或问题修复。

---

## 更新策略

- **主版本更新**：可能包含不兼容的更改，需要代码调整
- **次版本更新**：新增功能，向后兼容
- **补丁版本更新**：bug 修复，完全向后兼容

建议定期更新 SDK 到最新稳定版本以获得最佳性能和安全性。

## 总结

- **最低 Android 版本：** Android 6.0（API level 23）
- **支持平台：** Android 手机、平板、Android TV
- **支持语言：** Java、Kotlin
- **当前 SDK 版本：** v3.x
- **构建体系：** Gradle + AndroidX
